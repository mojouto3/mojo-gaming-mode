# Code of Conduct

## Our standard

Be respectful. Disagreements about code, features, or direction are fine
and expected, but keep it about the work, not the person.

Not okay, anywhere in this project (issues, pull requests, discussions,
commits, or the project's Discord):

- Harassment, insults, or personal attacks
- Discriminatory language or jokes
- Sharing others' private information without consent
- Sustained disruption of discussion (derailing, trolling)

## Scope

This applies to all spaces connected to the project: this repository
(issues, PRs, code review, commit messages), and any official community
spaces (such as the project's Discord server).

## Reporting

If someone is violating this, report it privately via Discord DM to the
maintainer. Reports are handled discreetly. Repeated or serious violations
result in removal from the project's spaces.

## Enforcement

The maintainer has final say on what does and doesn't fit this Code of
Conduct, and on what action to take. This can include a warning, temporary
restriction, or permanent ban from the project's spaces.
